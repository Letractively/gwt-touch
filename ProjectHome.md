### About ###
**gwt-touch** is a widget library to simplify tablet development when using Google Web Toolkit (GWT). Currently all touch libraries are pure JS (Sencha Touch, JQTouch, Dojox Mobile, etc). Our goal is to provide a pure GWT alternative.

This project kicked off recently, and already we have made a ton of great progress. Follow us on [twitter](http://twitter.com/bradrydzewski) for updates. Send us a message if you would like to [contribute](Contributing.md).
